鞋类产品保养学习感悟

  • 选购建议

    • 种类说明
      • 休闲鞋、硫化鞋、凉鞋、拖鞋不适合剧烈运动
      • 慢跑鞋、走路鞋不适合篮球、足球等剧烈活动
      • 室内运动鞋应用于室内
    • 选购方法
      • 试鞋时，应站立并双脚着地，以脚趾能上下活动为宜

  • 日常保养

    • 定期清洁鞋子

      • 对于鞋面，用棉布沾水轻轻擦，切勿猛刷
      • 对于清洗，浸泡时间<= 20min,洗涤温度 <= 45°
      • 对于晾晒，应放在阴凉通风处，避免阳光直射
      • 对于中底，普通污渍用橡皮擦，顽固污渍用软毛 +
      • 对于外底，及时清理沙石等杂物，用硬毛刷 + 中性洗涤剂清洗

    • 不要长期穿同一双鞋子

    • 鞋被雨水打湿后，应及时用干布吸干并放在通风处吹干

    • 鞋应避免接触酸、碱、油腐蚀物

    • 运动鞋适用的温度范围：-20°~45°

    • 鞋存放前，应清洗干净，再放于阴凉通风处存放

    • 对于底部有透气窗的鞋，应避免在雨天穿

    • 对于小白鞋，应在清洗后用白纸裹住晾晒

    • 对于鞋上的工艺，不要用硬毛刷清洗，防止脱落

    • 休闲鞋、板鞋不适合刷洗、机洗

    • 对于真皮皮鞋，普通污渍 / 灰尘，应先用湿布擦，再用皮革护理剂擦；若打湿，应及时吹干

    • 对于反绒皮的鞋，普通污渍 / 灰尘，应用软毛刷去除；可适当使用反绒皮护理剂，保护本色

    • 对于合成革的鞋，普通污渍 / 灰尘，应先用湿布擦；顽固污渍，可使用中性洗涤剂

  • 鞋的护理工具

    • 软毛刷
    • 反绒皮护理剂
    • 皮革护理剂
    • 棉布
    • 海绵
    • 防潮纸 / 干燥剂
    • 白色纸巾
